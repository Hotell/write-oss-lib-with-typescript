## Writing our library I

create our entry library point -> `index.ts` within `src` directory

`mkdir src`

`touch index.ts`

```ts
console.log('Hello World!');
```


GOOD JOB! NOW WHAT?

**Try it out !**

- compile your ts -> js

`yarn tsc`

`node lib/index.js`

You should see `"Hello World!"`

Also check `typings` folder -> we have our types generated
